Развлечение с магнитами. Решение

Магниты с фрагментами программы JavaScript перепутались на холодильнике. Удастся ли вам расставить магниты
по местам и восстановить работоспособную программу, чтобы она выдавала приведенный далее результат?
Возможно, какие-то магниты окажутся лишними. Ниже приведено наше решение.

function whatShallWear(temp){
	if (temp<60){
		console.log("Wear a jacket");
	} else if (temp<70){
		console.log("Wear a sweater");
	} else {
		console.log("Wear t-shirt");
	}
}
whatShallWear(50);
whatShallWear(80);
whatShallWear(60);
