А когда все будет сделано... Выполняется логика тела функции (в этом примере для имени Rover и веса
23 выводится сообщение "WOOF WOOF"), и работа функции на этом завершается. После этого управление
возвращается команде, следующей за вызовом bark.

function bark(name, weight){
	if (weight>20){
		console.log(name + " says WOOF WOOF");
	} else {
		console.log(name + " saysss woof woof");
	}
}

bark("rover", 23);
bark("spot", 13);
bark("spike", 53);
bark("lady", 17);
