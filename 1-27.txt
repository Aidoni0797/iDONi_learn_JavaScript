Как создаются команды

При создании контента HTML, вы обычно размечаете текст, определяя его структуру:
для этого в текст добавлюятся элементы, атрибуты и значения:

<h1 class="drink">Moccha Caffe Latte</h1>
<p>Espresso, steamed milk and chocolate syrup, just the way you like it.</p>

При работе с HTML мы размечаем текст для определения структуры: например, "так здесь у нас будет крупный
заголовок, а за ним следует абзац обычного текста"

с CSS дело обстоит немного иначе. Разработчик пишет набор правил: каждое правило выбирает элементы
страницы, а затем задает для этих элементов набор стилей:

h1.drink{
	color:brown;
}
p{
	font-family: sans-serif;
}

Для CSS пишутся правила с селекторами (например, h1.drink and p) определяющими, к каким частям разметки
HTML применяется данный стиль.

Допустим, все заголовки drink выводятся коричневым цветом.

Код JavaScript состоит из команд. Каждая команда описывает небольшую часть выполняемой операции, а весь
набор команд определяет поведение страницы:

var age=25;
var name = "Owen";

if (age>14) {
	alert("Sorry this page is for kids only!");
} else {
	alert("Welcome "+name+"!");
}
