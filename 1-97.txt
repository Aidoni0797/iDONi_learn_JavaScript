А теперь функцию нужно вызвать.

Чтобы вызвать функцию (передать ей управление), просто укажите ее имя, за которым в круглых скобках
перечисляются все передаваемые значения. Значения, передаваемые при вызове, называются аргументами. 
При вызове функции bark передаются два аргумента: имя собаки и ее вес.

Вот как происходит вызов:

#Имя функции bark
bark("rover", 23);
#"rover", 23 - передаются два аргумента: имя и вес собаки

Привызове функции bark аргументы связываются с именами параметров.

Каждый раз, когда имена параметров встречаются в функции, они заменяются переданными значениями.

После вызова вся основная работа выполняется в теле функции.

Когда значения всех параметров известны (например, параметр name содержит строку rover, 
а параметр weight число 23), можно переходить к выполнению тела функции.

Команды в теле функции выполняются сверху вниз, как и команды любого другого написанного вами кода.
Единственное отличие заключается в том, что с именами name and weight связываются значения аргументов,
передаваемые при вызове функции.

После того как значения аргументов будут связаны с именами параметров, начинается выполнение команд
в теле функции.

function bark(name, weight){
	if (23>20) {
		console.log("rover" + " says WOOF WOOF");
	} else {
		console.log("rover" + " says woof woof");
	}
}
#После того как значения аргументов будут связаны с именами параметров, начинается выполнение команд
в теле функции.
