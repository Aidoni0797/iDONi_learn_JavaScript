Допустим, вы подбираете машину для перемещений по Объектвилю. Какие использовать критерии? Как вам, например:
Год выпуска - 1960-й илил ранее.
Пробег - 10 000 миль и менее.
Чтобы задействовать новые навыки программирования (а заодно и упростить работу), вы хотите написать функцию
проверки машин. Если машина удовлетворяет критериям, фукнция вернет true, в противном случае на машину
можно не тратить время, а функция вернет false.

А если говорить конкретнее, мы напишем функцию, которая получает объект car в параметре, проверяет его и
возвращает булевское значение. Функция должна работать с любыми объектами car.

Давайте, попробуем:

Это функция
Функции будет предаваться объект car.
Для проверки свойств mileage(пробег) и year (год выпуска) достаточно применить точечную запись к параметру car.

Значение каждого из свойств проверяется по заданному критерию.

Если какая-либо из проверок не пройдена, функция возвращает false. В противном случае возвращается
true - проверка пройдена успешно!

funtion prequal(car){
	if (car.mileage>10000){
		return false;
	} else if (car.year > 1960) {
		return false;
	}
	return true;
}
