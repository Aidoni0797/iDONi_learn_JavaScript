Возьмите карандаш. Для каждого из приведенных ниже выражений вычислите его значение и запишите в выделенном
месте. Да, ЗАПИШИТЕ... забудьте слова мамы о том, что нельзя писать в книгах, и запишите ответ на странице!
Посмотрите наше решение.

(9/5)*temp+32
- Калькулятор пересчета температуры по Цельсию в шкалу Фаренгейта?

- Что случится, если переменная temp равна 10? 50

color == "orange"
- Логическое выражение. Оператор == проверяет, равна ли два значения.
- Истинным или ложным будет это выражение, если переменная color содержит значение "pink"? False
- А если значение "orange"? - true

name + ", " + "ypu've won!" 
- Какое значение будет, если переменная name содержит строку "Martha"? "Martha, you've won!"

yourLevel > 5
- Проверка: "первое значение больше второго?" Можно использовать оператор >= для условия "первое значение
больше либо равно второму?"
- Что получится, если переменная youLevel = 2? False
- Что получится, если переменная youLevel = 5? False
- Что получится, если переменная youLevel = 7? True

(level*points)+bonus 
- Переменная level = 5, points = 30000, bonus = 3300. Что получится при вычислении? 153300

color != "orange" - Если переменная color содержит строку "pink", это выражение будет истинным или ложным? true
- Оператор проверяет, что два значения не равны.

1000 + "108" - Вопрос на повышенную оценку!
- Существует несколько ответов. Из них правилен только один. Какой вы выберете? "1000108"

Для любознательных

Вы заметили, что в присваивании используется оператор =, а при проверке ==. Когжа вы присваеваете значение
переменной, ставьте один знак равенства =, а когда проверячете, совпадают ли два значения, - два
знака ==. Начинающие программисты часто путают эти операторы.
