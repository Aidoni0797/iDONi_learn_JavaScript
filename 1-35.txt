Возьми в руку карандаш

Возьмите карандаш. Для каждого из приведенных выражений вычислите его значение и запишите в выделенном месте.
Именно запишите..., забудьте слова мамы о том, что нельзя писать в книгах, и запишите ответ на странице!
Свериться с ответами можно в конце главы.

(9/5)*temp+32 - калькулятор пересчета температуры по Цельсиб в Фаренгейты?
Что получится, если переменная temp равна 10?

color == "orange" - логическое выражение. Оператор == проверяет, равны ли два значения.

name + ", " + "you've won!" - проверка: "первое значение больше второго?" Можно использовать оператор >=
для условия "первое значение больше либо равно второму?"

(level * points) + bonus

color != "orange" - оператор != проверяет, что два значения НЕ равны.

1000 + "108" - вопрос на повышенную оценку!
