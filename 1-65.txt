Пишем код упрощенной версии "Морского боя"

Псевдокод послужит нам "эскизом" для написания настоящего кода JavaScript. Для начала определимся с переменными.
Чтобы вспомнить, какие переменные нужны для программы "Морской бой", взгляните на псевдокод:

Объявить три переменные для хранения позиции каждой клетки корабля. Присвоить им имена 
locstion1, location2, location3.

Объявить переменную для номера текущей попытки. Присвоить ей имя quess.

Объявить переменную для количества попаданий. Присвоить ей имя hits и инициализировать значением 0.

Объявить переменную для количества попыток. Присвоить ей имя guesses и иницилизировать значением 0.

Объявить переменную для хранения информации о том, потоплен корабль или нет. Присвоить ей имя isSunk 
и инициализировать значением false.

Давайте определим эти переменные в файле JavaScript. Создайте новый файл с именем battleship.js и введите
следующие объявления переменных:

var location1 = 3;
var location2 = 4;
var location3 = 5;

var guess;
var hits = 0;
var guesses = 0;

var isSunk = false;

Для любознательных

Если вы не указали начальное значение переменной, JavaScript присваевает ей значение по умолчанию underfined
сообщая, что "переменная еще не инициализирована". Underifined и другие нестандартные значения позднее
будут рассмотрены более подробно.
