А если нужно принять много решений...

Вы можете объединить сколько угодно команд if/else. Также можно добавить завершающую секцию else;
если ни одно условие не выполняется, вы сможете обработать и эту ситуацию, как в следующем примере:

Сначала мы проверяем, осталось ли пять и более шариков мороженого...

if (scoops >= 5){
	alert("Eat faster, the ice cream is going to melt!");
} else if (scoops == 3) {
 	alert("Ice cream is runnong low!");
} else if (scoops == 2) {
	alert("Going once!");
} else if (scoops == 1) {
	alert("Going twice!");
} else if (scoops == 0) {
	alert("Gone!");
} else {
	alert("Still lots of ice cream left, come and get it.");
}
