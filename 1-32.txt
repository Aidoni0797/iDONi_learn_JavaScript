О синтаксисе

- Каждая команда завершается символом ";".
x = x + 1;

- Однострочный комментарий начинается с двух косых черт (//). Комментарии только содержит информацию о коде
для вас и других разработчиков. В программе они не выполяются
// Это комментарий

- Лишние пробелы разрешены (почти везде)
x    =    2233;

- Строки должны заключаться в двойные кавычки (или одиночные, но выберите что-то одно - будьте последовательны).

"You rule!"
'And so do you!'

- Логические значения true and false записываются без кавычек
rockin = true;

- Переменным не обязательно присваивать значение при объявлении:
var width;

- Язык JavaScript, в отличие от разметки HTML, учитывает регистр символов. Другими словами, 
Counter and counter - разные переменные.
