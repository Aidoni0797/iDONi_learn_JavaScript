Упражнение 5-6 "Декораторы"

Декоратор - приём программирования, который позволяет взять существующую функцию и изменить/расширить
ее поведение. Декоратор в нашем случае есть функция высшего порядка: получает функцию фактически в виде 
коллбэка и возвращает обработку, которая делает что-то своё "вокруг" вызова основной функции.
Декорируйте с помощью функции
const logged = function (f){
return function() {console.log(arguments); return f.apply(this, arguments)};
};
фнукцию
const qv = x=> x*x;
и вызовите так, чтобы значение первого аргумента было выведено в консоль
Примечание
ИСпользуйте то же имя, что и у декорируемой функции. Также см. декоратор @как bebel-плагин в главе 9.
